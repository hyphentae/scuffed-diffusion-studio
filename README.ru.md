# Бэкрумзинатор / Scuffed Diffusion Studio

[English](README.md) | **Русский**

Небольшое desktop-приложение на Python для перерисовки изображений с помощью Stable Diffusion image-to-image. Интерфейс построен на Tkinter, а модель загружается через Hugging Face Diffusers.

## Зачем создан проект

Я создал эту программу под впечатлением от фильма «Закулисье реальности». Мне понравилось, как визуальные элементы фильма напоминают изображения, которые были созданы ранними генеративными моделями ии. Поэтому я решил сделать инструмент, который намеренно искажает исходные изображения и придаёт им похожую атмосферу.

## Возможности

- выбор PNG, JPG, JPEG, WebP или BMP-файла;
- генерация вариации исходного изображения без текстового промпта;
- настройка силы изменения, guidance scale и количества проходов;
- просмотр исходного и получившегося изображения;
- сохранение результата в `output.png`;
- экспорт полученной картинки в выбранный пользователем файл PNG, JPEG или WebP.

## Требования

- Python 3.9 или новее;
- Tkinter;
- GPU с CUDA рекомендуется, но приложение может работать на CPU;
- подключение к интернету при первом запуске для скачивания модели `CompVis/stable-diffusion-v1-1`.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate       # Windows: .venv\Scripts\activate
pip install torch torchvision diffusers transformers Pillow
```

На Windows и Linux способ установки `torch` может зависеть от версии CUDA. При необходимости установите PyTorch по инструкции на [pytorch.org](https://pytorch.org/), затем выполните установку остальных зависимостей.

## Запуск

```bash
python main.py
```

После запуска приложение автоматически начнёт загрузку модели. Когда модель будет готова:

1. Нажмите «Выбрать картинку».
2. Настройте параметры генерации.
3. Нажмите «Перерисовать».
4. Нажмите «Сохранить результат» и выберите путь и формат файла.

Исходное изображение сохраняется как `input.png`, результат — как `output.png` в текущей папке. Кнопка «Сохранить результат» экспортирует полученную картинку в PNG, JPEG или WebP. Кнопка «Удалить файлы» удаляет оба файла и очищает превью.

## Параметры

- **Strength** (`0–1`) — степень изменения исходного изображения. Большие значения дают более сильную перерисовку.
- **Guidance scale** — степень влияния текстового условия. В текущей версии промпт пустой, поэтому параметр оставлен как настройка пайплайна.
- **Steps** — количество шагов генерации. Большее значение обычно увеличивает время работы.

Приложение изменяет размер изображения до `512×512` перед генерацией. Модель инициализируется без safety checker (`safety_checker=None`); используйте приложение ответственно.
